(protected) onClose(ev)

A 'close' message has been received, we MUST close. If we have inner containers, send them a 'close', then once inners unloaded then close. If was a message bubbled up from a inner container (inner container did not respond to the close) So clear it for them, then trigger a 'unload' on behalf of the inner.
